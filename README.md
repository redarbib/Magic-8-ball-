# Online Magic 8-Ball+

A simple (but fun) web app where you ask a question and get a random answer  with different vibes (Funny / Serious / Savage), a confidence, and optional local history.

## Features
- Ask a question  get a random answer
- Modes: Funny, Serious, Savage
- Random confidence score
- Optional: save the last 10 questions locally (LocalStorage)
- Clear history button

## Tech
- HTML + CSS + Vanilla JavaScript
- No backend required
